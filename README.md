# NixOS Laptop Configuration

A modular, flake-based NixOS configuration for a laptop featuring KDE Plasma 6 on Wayland, Btrfs with LUKS encryption, TPM auto-unlock, and a clean separation between public config and private machine settings.

## Features

- **Desktop**: KDE Plasma 6 (Wayland) with plasma-manager for declarative settings.
- **Filesystem**: Btrfs with subvolumes, weekly scrubbing, and zstd compression.
- **Security**: Full disk encryption (LUKS) with TPM2 auto-unlock and fingerprint reader support.
- **Apps**: Firefox, Chrome, VSCode, Spotify, Bitwarden, Ghostty, Steam, Podman, Signal, etc.
- **Management**: Home Manager, Flakes, auto-upgrades, garbage collection.
- **Secrets**: Managed via sops-nix.
- **Performance**: Zen kernel, ananicy-cpp, irqbalance, power-profiles-daemon, amd_pstate=active.

> **Hardware assumption**: This config is tailored for an AMD system. The Zen kernel and `amd_pstate=active` kernel parameter are AMD-specific. Intel users should change `linuxPackages_zen` to `linuxPackages_latest` and remove `amd_pstate=active` from `modules/nixos/common.nix`.

## Structure

```
flake.nix                              # Flake entry point
settings.nix.example                   # Template for your private settings
hosts/laptop/
  default.nix                          # Host config
  hardware-configuration.nix           # Machine-specific (in private repo, not committed)
modules/nixos/
  common.nix                           # Boot, kernel, power, TPM, fingerprint, tailscale
  kde.nix                              # KDE Plasma, SDDM, system packages
  flatpak.nix                          # Declarative flatpak management
  secrets.nix                          # sops-nix secrets
modules/home-manager/
  default.nix                          # User packages, git config
  kde-settings.nix                     # plasma-manager settings (panel, kwin, dolphin)
```

## Private Repo

Machine-specific files (`settings.nix` and `hardware-configuration.nix`) live in a separate private repo to keep sensitive details out of this public repo. The private repo is referenced via the nix registry.

**1. Create your private repo** containing:
- `settings.nix` — copy from `settings.nix.example` and fill in your values
- `hardware-configuration.nix` — generated by `nixos-generate-config`

**2. Update the private input URL in `flake.nix`:**
```nix
private = {
  url = "git+ssh://git@github.com/youruser/your-private-repo";
  flake = false;
};
```

## Installation

### 1. Partition and Encrypt

Boot into the NixOS Live ISO.

```bash
# Partition (use cfdisk or gdisk)
cfdisk /dev/nvme0n1
# 512M EFI partition (type EF00)
# Remaining space as Linux filesystem (type 8300)

# Format boot
mkfs.vfat -F32 -n boot /dev/nvme0n1p1

# Encrypt root — the name "cryptroot" is expected by the config, keep it as-is
cryptsetup luksFormat /dev/nvme0n1p2
cryptsetup open /dev/nvme0n1p2 cryptroot

# Format Btrfs
mkfs.btrfs -L nixos /dev/mapper/cryptroot
mount /dev/mapper/cryptroot /mnt

# Create subvolumes
btrfs subvolume create /mnt/@
btrfs subvolume create /mnt/@home
btrfs subvolume create /mnt/@nix
btrfs subvolume create /mnt/@log
umount /mnt

# Mount subvolumes
mount -o compress=zstd,subvol=@ /dev/mapper/cryptroot /mnt
mkdir -p /mnt/{boot,home,nix,var/log}
mount -o compress=zstd,subvol=@home /dev/mapper/cryptroot /mnt/home
mount -o compress=zstd,subvol=@nix /dev/mapper/cryptroot /mnt/nix
mount -o compress=zstd,subvol=@log /dev/mapper/cryptroot /mnt/var/log
mount /dev/nvme0n1p1 /mnt/boot
```

### 2. Set Up Private Repo and Registry

Generate your hardware config:
```bash
nixos-generate-config --root /mnt --show-hardware-config
```

Add the output to your private repo as `hardware-configuration.nix`, along with your filled-in `settings.nix`.

### 3. Install

```bash
git clone https://github.com/youruser/nixlap /mnt/etc/nixos
cd /mnt/etc/nixos
nixos-install --flake .#laptop
```

> `system.stateVersion` is in `hosts/laptop/default.nix` and `home.stateVersion` is in `modules/home-manager/default.nix`. Both are set to `"25.11"` — update them to your current NixOS release if installing fresh.

### 4. Post-Install

#### TPM Auto-Unlock

```bash
sudo systemd-cryptenroll --tpm2-device=auto --tpm2-pcrs=0+2 /dev/nvme0n1p2
```

#### Secrets with sops-nix

```bash
sudo mkdir -p /var/lib/sops-nix
sudo age-keygen -o /var/lib/sops-nix/key.txt
sops secrets/secrets.yaml
```

Uncomment the sops configuration in `modules/nixos/secrets.nix`.

#### Flatpaks

Managed declaratively in `modules/nixos/flatpak.nix`.
